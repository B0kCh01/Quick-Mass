# Quick-Mass
Fancy Python Molar Mass calculator

## Features
- Tutorial
- Smart Error Check
- Scrolling Text Interface
- Generated Math Expression
- Precise Calculator

## Running the Program
- __Run on Repl.it (recommended)__
  - https://bit.ly/quickMass
- __Run natively__
  - Make sure to have the following libraries with pip
    1. time
    2. pyfiglet
    3. termcolor
    4. peridictable
    5. RegEx
